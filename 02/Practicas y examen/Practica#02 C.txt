Practica 2

2)Boton que permita al usuario crear la matriz con los tamaños especificados.
3)Textbox y boton para introducir valores a la matriz.
4)Labels que se actualicen mostrando la fila y la columna donde se va a escribir a continuacion.
5)Que el programa no permita meter mas valores a la matriz una vez que ya este llena.

Una vez creada la matriz hacer las siguientes funciones:
7)Boton que calcule el promedio de la matriz.
8)El usuario pueda calcular el promedio de una fila o columna.
9)El usuario puede obtener el valor maximo.
10)El usuario puede obtener el valor minimo.
11)Boton para mostrar la matriz.
